1. Click the green "<> Code V" button. Click Codespaces, click "Create codespace on main".
2. Now that you are in the codespace, you will see a message at the bottom right that says "Building Codespace", wait for it to go away.
3. Wait for an elephant icon to appear on the sidebar.
4. Wait for the loading circle on the bottom bas next to the antenna to go away.
5. You are now ready to code!
6. Your code is stored in lib/src/main/GoogleBillboard.java
7. To test your code, click the elephant in the sidebar, and navigate to lib > Tasks > build > testClasses.
8. Running the testClasses task will result in an ERROR! This is because you have failing test cases, edit your code to fix them!
9. When all tests are passing, go to the Souce Control tab in the sidebar, beneath the search, add a descriptave message, stage all your changes, commit and push!
